  # AI-Powered Carbon Credit Verification Platform

Hackathon MVP demonstrating real-time AI reasoning for carbon credit verification.

## Tech Stack
- Frontend: HTML, CSS, JavaScript
- Backend: Firebase (Auth + Firestore)
- AI: Google Gemini API
- Maps: Google Maps API

## Setup
1. Clone repository
2. Configure Firebase credentials in `js/firebase-config.js`
3. Add Gemini API key in `js/gemini-api.js`
4. Open `index.html` in browser
